# Silksong Monect Controller Layout

Custom controller layout for Hollow Knight: Silksong using Monect PC Remote.

## Features
- Optimized combat controls
- Easy movement
- Mobile friendly layout

## How to Use
1. Download layout file
2. Import into Monect
3. Start the game
